🚀 HOW TO RUN (3 STEPS):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Open 3 New Terminal Windows

STEP 2: Run Each in a Different Terminal:

  TERMINAL 1 - Backend (Port 3001):
  ├─ cd c:/amd/analytics-platform/backend
  ├─ npm install
  └─ npm run dev
  
  TERMINAL 2 - Frontend (Port 3000):
  ├─ cd c:/amd/analytics-platform/frontend
  ├─ npm install
  └─ npm start
  
  TERMINAL 3 - ML Service (Port 5000):
  ├─ cd c:/amd/analytics-platform/ml-service
  ├─ python -m venv venv
  ├─ venv\Scripts\activate.bat (Windows) OR source venv/bin/activate (Mac/Linux)
  ├─ pip install -r requirements.txt
  └─ python app.py

STEP 3: Open Browser & Visit
  └─ http://localhost:3000
